# 个人主页需求文档

## 1. 应用概述

### 1.1 应用名称
张亦弛的个人主页

### 1.2 应用描述
一个展示个人 profile 和提供数字分身聊天互动的个人主页，面向朋友、AI同行者和面试官，帮助访客快速了解本人信息并进行互动问答。

## 2. 核心功能

### 2.1 个人信息展示区
- 头像展示
- 基本信息：
  - 名字：张亦弛
  - 一句话介绍：一个正在自学 AI 并希望能找到相关实习工作的研一学生
  - 当前学习内容：学习 openclaw，学习 vibe coding，学习 AI 基础知识，学习制作产品
  - 兴趣方向：AI 应用、旅行、运动
  - 特色描述：希望能最终实现财务自由以获得身体自由

### 2.2 数字分身聊天区
一个可与访客互动的数字分身模块，需预设以下知识：
- 职业身份：一个正在自学 AI 并希望能找到相关实习工作的研一学生
- 最近在做的事：学习 openclaw，学习 vibe coding，学习 AI 基础知识，学习制作产品
- 擅长/关心的方向：AI 应用、AI agent，如何学习 AI 以满足客户需求找到工作
- 常见问题回答：
  - 你现在在做什么？→ 回答当前学习内容和项目
  - 怎么联系你？→ 提供联系方式
  - 其他相关问题的对应回答

## 3. 页面结构

### 3.1 主要内容区域
- 顶部：标题/欢迎语
- 中部：个人信息展示区
- 下部：数字分身聊天区

### 3.2 适配要求
- 支持手机端适配
- 信息层次清晰，避免花哨设计

## 4. 设计要求

### 4.1 风格
- 简约、清爽、科技感
- 颜色偏好：深蓝色和白色

### 4.2 其他
- 界面需清晰明了，聚焦核心信息展示
- 保持整体风格统一
